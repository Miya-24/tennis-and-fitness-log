# 📖 リポジトリ運用マニュアル

このリポジトリは、テニスの上達と体重58kg達成に向けた「PDCAサイクル」を回すためのツールです。

## 🔄 基本の運用サイクル

### 1. 記録（Input）
- **場所**: ジム、テニスコート、移動中
- **アクション**: 
  - `Issues` から `New Issue` を作成。
  - 内容に合わせて `Training / Diet / Tennis Log` テンプレートを選択して入力・投稿。

### 2. 分析・対話（Analysis）
- **場所**: 自宅（大画面モニター前推奨）
- **アクション**:
  - Issueの内容をGeminiに貼り付け「コーチ陣を召喚」してディスカッションを開始。
  - **滝沢（根性）、エミリ（科学）、白石（感覚）、影山（個人戦術）、サラ（団体戦術）**からフィードバックを受ける。

### 3. 整理・集約（Weekly Archive）
- **タイミング**: 週に1回（週末）
- **アクション**:
  - その週のIssueを振り返り、`/summaries/` フォルダに週間まとめ（YYYY-MM-DD.md）を作成。
  - **個別の詳細mdは作成せず**、この週間まとめがリポジトリの正史となる。
  - 保存後、対象のIssueをすべて `Close` する。

### 4. アップデート（Update）
- **アクション**:
  - `README.md` の体重（進捗バー）を更新。
  - コーチ陣との対話で決まった「新しい課題」を目標リストに追加。
