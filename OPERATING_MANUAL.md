# 📖 リポジトリ運用マニュアル

このリポジトリは、テニスの上達と体重58kg達成に向けた「PDCAサイクル」を回すためのツールです。

## 🔄 基本の運用サイクル

### 1. 記録（Input）
- **場所**: ジム、テニスコート、移動中
- **ツール**: スマホ
- **アクション**: 
  - `Issues` から `New Issue` を作成。
  - 内容に合わせて `Training Log` または `Tennis Log` テンプレートを選択。
  - 数値や気づきを入力して `Submit`。

### 2. 分析・対話（Analysis）
- **場所**: 自宅
- **ツール**: Gemini (AIコーチ陣)
- **アクション**:
  - Issueの内容、またはアーカイブしたmdの内容をコピー。
  - Geminiに「コーチ陣を召喚」と伝え、内容を貼り付けてディスカッションを開始。
  - **滝沢（根性）、エミリ（科学）、白石（感覚）、影山（個人戦術）、サラ（団体戦術）**の5視点からフィードバックを受ける。

### 3. 整理・保存（Archive）
- **タイミング**: 週に1回、またはIssueが溜まった時
- **アクション**:
  - Issueの内容を、適切なディレクトリ（`/fitness/gym/` や `/tennis/log/`）に `.md` ファイルとして保存。
  - ファイル名は `YYYY-MM-DD_内容.md` とする。
  - 完了したIssueは `Close` する。

### 4. アップデート（Update）
- **アクション**:
  - `README.md` の「Current Status」の体重や進捗バーを更新。
  - コーチ陣との対話で決まった「新しい課題」を `README.md` の目標リストに追加。
